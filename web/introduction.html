Welcome to StatMediaWiki home site

StatMediaWiki is a project that aims to create a tool to collect and aggregate information available in a MediaWiki installation to monitor its development.

Results are static HTML pages including tables and graphics. StatMediaWiki is free software under the GPL v3 or higher license.

Some functionalities in StatMediaWiki are adapted from those in StatSVN.
